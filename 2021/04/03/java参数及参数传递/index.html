д»ҠеӨ©еҲ·зүӣе®ўзҪ‘ Java дё“йЎ№д№ йўҳзҡ„ж—¶еҖҷзў°еҲ°дәҶдёҖйҒ“е…ідәҺ Java еҸӮж•°еҸҠеҸӮж•°дј йҖ’е’ҢStringзұ»еһӢж•°жҚ®дҪңдёәеҸӮж•°дј йҖ’зҡ„жғ…еҶөпјҢеӣһйЎҫдәҶдёҖдёӢзҹҘиҜҶзӮ№пјҢзҺ°еңЁеҒҡдёҖдёӢз®ҖеҚ•зҡ„жҖ»з»“гҖӮ

еҪўејҸеҸӮж•°е’Ңе®һйҷ…еҸӮж•°

еңЁзЁӢеәҸиҜӯиЁҖи®ҫи®ЎдёӯпјҢеҸӮж•°жҳҜжңүеҪўејҸеҸӮж•°е’Ңе®һйҷ…еҸӮж•°д№ӢеҲҶзҡ„пјҢе…ҲзңӢдёҖдёӢеҪўеҸӮе’Ңе®һеҸӮзҡ„е®ҡд№үпјҡ

еҪўејҸеҸӮж•°пјҡжҳҜеңЁе®ҡд№үеҮҪж•°еҗҚе’ҢеҮҪж•°дҪ“зҡ„ж—¶еҖҷдҪҝз”Ёзҡ„еҸӮж•°пјҢзӣ®зҡ„жҳҜз”ЁжқҘжҺҘж”¶и°ғз”ЁиҜҘеҮҪж•°ж—¶дј е…Ҙзҡ„еҸӮж•°пјҢз®Җз§°вҖңеҪўеҸӮвҖқгҖӮ

е®һйҷ…еҸӮж•°пјҡеңЁдё»и°ғеҮҪж•°дёӯи°ғз”ЁдёҖдёӘеҮҪж•°ж—¶пјҢеҮҪж•°еҗҚеҗҺйқўжӢ¬еҸ·дёӯзҡ„еҸӮж•°з§°дёәвҖңе®һйҷ…еҸӮж•°вҖқпјҢз®Җз§°вҖңе®һеҸӮвҖқгҖӮ

дёҫдёӘдҫӢеӯҗпјҡ

Personзұ»дёӯе®ҡд№үдәҶдёҖдёӘsoutж–№жі•пјҢиҜҘж–№жі•жңүдёӘStringзұ»еһӢзҡ„еҸӮж•°nameпјҢиҜҘеҸӮж•°еҚідёәеҪўеҸӮгҖӮеңЁmainж–№жі•дёӯпјҢи°ғз”ЁдәҶsoutж–№жі•пјҢдј е…ҘдәҶдёҖдёӘеҸӮж•°вҖңеј дёүвҖқпјҢиҜҘеҸӮж•°еҚідёәе®һеҸӮгҖӮ

й—®йўҳжқҘдәҶпјҡе®һйҷ…еҸӮж•°жҳҜеҰӮдҪ•дј е…Ҙж–№жі•зҡ„е‘ўпјҹ

еҖјдј йҖ’е’Ңеј•з”Ёдј йҖ’

еҸӮж•°дј йҖ’зҡ„жңәеҲ¶жңүдёӨз§ҚпјҡеҖјдј йҖ’е’Ңеј•з”Ёдј йҖ’пјҢе…ҲдёҠе®ҡд№үпјҡ

еҖјдј йҖ’пјҡжҳҜжҢҮеңЁи°ғз”ЁеҮҪж•°ж—¶е°Ҷе®һйҷ…еҸӮж•°еӨҚеҲ¶дёҖд»Ҫдј йҖ’еҲ°еҮҪж•°дёӯпјҢиҝҷж ·еңЁеҮҪж•°дёӯеҰӮжһңеҜ№еҸӮж•°иҝӣиЎҢдҝ®ж”№пјҢе°ҶдёҚдјҡеҪұе“ҚеҲ°е®һйҷ…еҸӮж•°гҖӮ

еј•з”Ёдј йҖ’пјҡжҳҜжҢҮеңЁи°ғз”ЁеҮҪж•°ж—¶е°Ҷе®һйҷ…еҸӮж•°зҡ„ең°еқҖдј йҖ’еҲ°еҮҪж•°дёӯпјҢйӮЈд№ҲеңЁеҮҪж•°дёӯеҜ№еҸӮж•°жүҖиҝӣиЎҢзҡ„дҝ®ж”№пјҢе°ҶеҪұе“ҚеҲ°е®һйҷ…еҸӮж•°гҖӮ

еҹәжң¬ж•°жҚ®зұ»еһӢдҪңдёәеҸӮж•°дј йҖ’

еҹәжң¬ж•°жҚ®зұ»еһӢдҪңдёәеҸӮж•°дј йҖ’ж—¶йғҪжҳҜдј йҖ’еҖјзҡ„жӢ·иҙқгҖӮ еҚіеңЁи°ғз”Ёж–№жі•ж—¶пјҢе°ҶиҰҒдј йҖ’зҡ„еҖјпјҲе®һеҸӮпјүжӢ·иҙқдёҖд»Ҫдј е…Ҙж–№жі•пјҢеӣ жӯӨж— и®әж–№жі•жҖҺд№Ҳж”№еҸҳдј е…Ҙзҡ„еҖјйғҪдёҚдјҡеҪұе“Қе®һйҷ…еҸӮж•°зҡ„еҖјгҖӮ

з®ҖеҚ•ең°иҜҙе°ұжҳҜжҠҠе®һеҸӮзҡ„жӢ·иҙқеҖјиөӢз»ҷеҪўеҸӮпјҢеҜ№еҪўеҸӮзҡ„ж”№еҸҳдёҚдјҡеҪұе“Қе®һеҸӮзҡ„еҖјгҖӮ

иҫ“еҮәз»“жһңдёәпјҡ

еҜ№иұЎдҪңдёәеҸӮж•°дј йҖ’

еҜ№иұЎдҪңдёәеҸӮж•°дј йҖ’ж—¶пјҢеңЁж–№жі•еҶ…ж”№еҸҳеҜ№иұЎзҡ„еҖјпјҢжңүж—¶еҺҹеҜ№иұЎи·ҹзқҖж”№еҸҳпјҢиҖҢжңүж—¶еҸҲжІЎжңүж”№еҸҳгҖӮе…ҲзңӢдҫӢеӯҗпјҡ

иҫ“еҮәз»“жһңдёәпјҡ

йҖ жҲҗиҜҘз»“жһңзҡ„еҲҶжһҗеҰӮдёӢпјҡ

з”ұжӯӨеҸҜи§ҒпјҢеңЁjavaдёӯеҜ№иұЎдҪңдёәеҸӮж•°дј йҖ’ж—¶пјҢдј йҖ’зҡ„жҳҜеј•з”Ёзҡ„ең°еқҖпјҢжҳҜжҠҠеҜ№иұЎеңЁеҶ…еӯҳдёӯзҡ„ең°еқҖжӢ·иҙқдәҶдёҖд»Ҫдј з»ҷдәҶеҸӮж•°гҖӮжүҖд»ҘдёәдәҶж–№дҫҝзҗҶи§ЈпјҢеҸҜд»ҘиҜҙеңЁjavaдёӯеҸӘжңүеҖјдј йҖ’жІЎжңүеј•з”Ёдј йҖ’пјҢеӣ дёәеј•з”Ёдј йҖ’еҪ’ж №з»“еә•д№ҹжҳҜдј йҖ’ең°еқҖзҡ„жӢ·иҙқеҖјгҖӮ

3.Stringзұ»еһӢж•°жҚ®дҪңдёәеҸӮж•°дј йҖ’

еҹәжң¬ж•°жҚ®зұ»еһӢзҡ„еҢ…иЈ…зұ»еһӢеңЁдј йҖ’еҸӮж•°ж—¶е…¶е®һд№ҹжҳҜвҖңжҢүеј•з”Ёдј йҖ’зҡ„вҖқпјҢеҸӘжҳҜеӣ дёәеҢ…иЈ…зұ»еһӢеҸҳйҮҸйғҪжҳҜдёҚеҸҜеҸҳйҮҸпјҢе®№жҳ“иҜҜи§ЈгҖӮ

StringжҳҜfinalзұ»еһӢпјҢжҳҜдёӘзү№ж®Ҡзҡ„зұ»пјҢеҜ№е®ғзҡ„дёҖдәӣж“ҚдҪңз¬ҰжҳҜйҮҚиҪҪзҡ„гҖӮжҜ”еҰӮпјҡ

з”ұжӯӨе®№жҳ“зҗҶи§Јд»ҘдёӢд»Јз Ғзҡ„иҫ“еҮәз»“жһңпјҡ

ж–№жі•и°ғз”Ёж—¶пјҢеҗҚз§°зӣёеҗҢзҡ„е®һеҸӮе’ҢеҪўеҸӮе№¶дёҚдёҖж ·пјҢдёҖдёӘжҳҜmain()дёӯзҡ„str,жҢҮеҗ‘еӯҳж”ҫвҖқhelloвҖқзҡ„еҶ…еӯҳең°еқҖгҖӮдёҖдёӘжҳҜfun()дёӯзҡ„strпјҢstr=вҖқworldвҖқ,зӣёеҪ“дәҺnew String(вҖңworldвҖқ)гҖӮ==StringжҳҜfinalзұ»еһӢпјҢе°ҶеңЁе ҶдёӯйҮҚж–°еҲҶй…ҚдёҖдёӘеҶ…еӯҳз©әй—ҙеӯҳж”ҫвҖқworldвҖқ==гҖӮch[0]=вҖҷdвҖҷ,еҜ№иұЎзҡ„еҶ…е®№еҸ‘з”ҹж”№еҸҳгҖӮ
